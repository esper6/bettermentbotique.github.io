Overview

This privacy policy describes how the Social Media Content Generator ("the App") handles user data.

Data Collection

The App only accesses:

  • YouTube upload capabilities through user-authorized OAuth
  • Instagram upload capabilities through user-authorized OAuth
  • Local video and image files specified by the user

Data Usage

The App uses this access solely to:

  • Upload videos to the user's YouTube channel
  • Upload videos to the user's Instagram account
  • No data is stored on external servers
  • No user data is shared with third parties

Data Storage

The App stores:

  • OAuth tokens locally for authentication
  • Temporary video and image files in user-specified directories
  • All data is stored locally on the user's machine

User Rights

Users can:

  • Revoke access through YouTube/Instagram settings
  • Delete all local data by removing the application files

Data Deletion

To request data deletion:

  • Revoke app access in your YouTube settings
  • Revoke app access in your Instagram settings
  • Delete the local application directory
